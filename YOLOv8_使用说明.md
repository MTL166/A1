# YOLOv8目标检测界面使用说明

## 功能概述

基于Django框架开发的YOLOv8目标检测Web界面，具备以下功能：

1. ✅ 用户登录验证（未登录无法访问检测界面）
2. ✅ 图片上传和检测
3. ✅ 视频上传和检测（返回第一帧检测结果）
4. ✅ 检测结果实时显示
5. ✅ 与其他界面一致的登录/登出功能
6. ✅ 界面左上角返回首页功能

## 访问方式

1. 启动Django服务器：
   ```bash
   python manage.py runserver
   ```

2. 访问首页：http://127.0.0.1:8000/

3. 从首页点击"YOLOv8目标检测"链接进入检测界面

## 使用步骤

1. **登录系统**
   - 如果未登录，会自动跳转到登录页面
   - 使用有效的用户名和密码登录

2. **选择检测类型**
   - **图片检测**：上传图片文件（支持jpg、png等格式）
   - **视频检测**：上传视频文件（支持mp4、avi等格式）

3. **查看检测结果**
   - 检测完成后，界面会显示：
     - 标注后的图像（包含检测框和标签）
     - 检测到的目标详细信息（类别、置信度、位置坐标）

## 文件结构

```
myweb/
├── myapp/
│   ├── yolov8_utils.py      # YOLOv8工具函数
│   ├── views.py             # 视图函数（新增检测功能）
│   ├── urls.py              # URL路由（新增检测路由）
│   └── decorators.py        # 登录验证装饰器
├── templates/
│   └── myapp/
│       └── yolov8/
│           └── detection.html  # 检测界面模板
├── yolov8n.pt              # YOLOv8模型文件
└── test_yolov8.py          # 功能测试脚本
```

## 依赖包

确保安装以下Python包：
```bash
pip install ultralytics opencv-python
```

## 注意事项

1. **模型文件**：确保`yolov8n.pt`文件存在于项目根目录
2. **文件大小**：大文件上传可能需要较长时间处理
3. **首次加载**：第一次加载模型可能需要一些时间
4. **硬件要求**：需要足够的GPU/CPU资源进行目标检测

## 故障排除

如果遇到问题，可以运行测试脚本验证功能：
```bash
python test_yolov8.py
```
